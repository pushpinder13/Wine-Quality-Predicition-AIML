# Wine Quality Prediction

Predicts wine quality using machine learning on a dataset of 1599 red wine samples with 11 features.

## Dataset
- **File**: winequality-red.csv
- **Size**: 1599 data points
- **Features**: 11 chemical properties
- **Target**: Quality score (3-9)

## Features
- fixed acidity
- volatile acidity
- citric acid
- residual sugar
- chlorides
- free sulfur dioxide
- total sulfur dioxide
- density
- pH
- sulphates
- alcohol

## Quality Categories
- **Bad**: Quality < 6
- **Average**: Quality = 6
- **Good**: Quality > 6

## Model
- **Algorithm**: Random Forest
- **Training Accuracy**: ~93%
- **Technology**: Python3
